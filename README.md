# Air Indo - Airlines Ticket Platform

A modern, responsive airline ticket booking platform built with React and Tailwind CSS.

## Features

- 🎨 Beautiful Air Indo theme and color system
- ✈️ Flight search and booking interface
- 🎯 Responsive design for all devices
- 🔐 Secure user authentication
- 💳 Multiple payment options
- 🌍 Global destination coverage

## Theme & Colors

The platform uses a professional airline color scheme:

- **Primary Dark**: #003d52 (Navy Blue)
- **Primary Blue**: #1a73b9 (Sky Blue)
- **Accent Gold**: #ffd769 (Gold)
- **Accent Brown**: #d4a574 (Warm Brown)
- **Teal**: #4db8ab (Turquoise)

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/          # Reusable React components
│   ├── Header.jsx       # Navigation header
│   ├── Hero.jsx         # Hero section
│   └── SearchForm.jsx   # Flight search form
├── styles/              # Global styles and theme
│   ├── globals.css      # Global CSS with Tailwind directives
│   └── colors.js        # Color system export
├── App.jsx              # Main app component
└── main.jsx             # React entry point
```
